# ⚡ TokenPilot — AI Prompt Helper

**TokenPilot** is a premium Chrome extension designed to improve your workflow across all major AI platforms (ChatGPT, Claude, Gemini, Perplexity, and more). It provides real-time token estimation, readability analysis, and a local prompt history.

---

## 🚀 Key Features

- **Global Context Support**: Run on any AI platform (including Arena.ai) and standard text inputs.
- **Real-Time Token Counter**: Accurate estimation for GPT-4, Claude, and Gemini models.
- **Dynamic Platform Info**: Displays specific free-tier usage limits (e.g., *~10 msgs / 5h* for ChatGPT) based on the site you are on.
- **Readability Analysis**: Shows a "Simple", "Moderate", or "Complex" badge based on your prompt structure to help you get better AI results.
- **Local Prompt History**: Automatically saves your most recent prompts (Enter key or on "Send") for quick restoration.
- **Premium Interface**: A sleek, minimal glassmorphism UI that can be collapsed when not needed.

---

## 🛠️ Installation Instructions

Since this is a pro tool in development, follow these steps to install it in your browser:

1. **Download or Clone**: Download this repository to a folder on your computer.
2. **Open Extensions Page**: In Chrome (or Edge/Brave), go to `chrome://extensions`.
3. **Enable Developer Mode**: Flip the "Developer mode" toggle in the top-right corner.
4. **Load Unpacked**: Click the **"Load unpacked"** button and select the folder containing the TokenPilot files.
5. **Activity**: Open any AI site (like ChatGPT) and look for the ⚡ widget in the bottom-right corner!

## 🔒 Privacy & Security

TokenPilot is designed with privacy in mind:
- **No External API Calls**: Does not send your data to any third-party services.
- **Local Storage**: Your prompt history is stored only on your computer (`localStorage`).
- **Zero Configuration**: No API keys or accounts required.
